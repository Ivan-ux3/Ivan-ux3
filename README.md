## Projetos

### CUT! - Plataforma de Streaming e Rede Social
Plataforma web full stack que integra streaming de filmes e séries com elementos de rede social.  
Usuários podem publicar resenhas, avaliar títulos, curtir comentários e acompanhar um feed personalizado.  
**Tecnologias:** React, Vite, HTML5, CSS3, JavaScript, Node.js, Express, Firebase  
[Ver no GitHub](https://github.com/Ivan-ux3/projeto-cut)

### Crypto Lab - Laboratório de Criptografia Educacional
Laboratório web para aplicar diferentes métodos clássicos de criptografia em textos, com interface integrada a uma API REST.  
Permite aplicar múltiplas cifras em sequência e serve como projeto educacional de integração frontend/backend.  
**Tecnologias:** React, JavaScript, CSS, Python, FastAPI, Pydantic, Uvicorn  
[Ver no GitHub](https://github.com/Ivan-ux3/crypto_lab)

### Projeto Senhas - Cofre de Senhas Seguro
Aplicação em Python para gerenciamento seguro de senhas, com cofre protegido por senha-mestra.  
Permite gerar, armazenar e consultar senhas criptografadas, além de verificar vazamentos públicos via API Have I Been Pwned.  
**Tecnologias:** Python 3, Git  
[Ver no GitHub](https://github.com/Ivan-ux3/projeto_senhas)
